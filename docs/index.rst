Usage
=====

Installation
------------

.. code-block:: console

   $ git clone https://github.com/dice-group/dice-embeddings.git

   $ conda create -n dice python=3.10 --no-default-packages && conda activate dice

   (dice) $ pip3 install -r requirements.txt

or

.. code-block:: console

   (dice) $ pip install dicee
